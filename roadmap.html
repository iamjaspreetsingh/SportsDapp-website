The Strategic Plan


12 November 2019

INTERNAL TEST RELEASE
The mobile application is released for internal testings on all the live matches.
Anyone can join the program providing us the email.

24 November 2019

ALPHA VERSION RELEASE
Alpha version of app is released for feedback response of users playing the game and testing it in real environment.
The reward model would be based on App points and Win points that can be cashed in later. No one would be requiring to invest money.

BETA VERSION RELEASE
Beta version of app is released after scaling up the solution and marketing the application.
The reward model would also include investments from users hence multi scaling the cash rewards collected transparently.

PRODUCTION LAUNCH RELEASE
Application is made available to everyone worldwide taking care of user experience and user guides.
